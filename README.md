# pyqt-googlemap-example
PyQt window which shows the Google map

## How To Use
1. git clone ~
2. python main.py

## Prerequisites
python -m pip install PyQtWebEngine

python -m pip install PyQt5-stubs - if this is not installed, QtWebEngineWidgets does not work
